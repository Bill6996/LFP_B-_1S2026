DESCRIPCION DE MI PROYECTO DE LENGUAJES FORMALES UNICA PRACTICA!!
Sistema de consola desarrollado en C++ para el análisis de rendimiento académico universitario. El programa lee tres archivos .lfp con datos de estudiantes, cursos y calificaciones, los procesa mediante separación por comas, calcula estadísticas descriptivas avanzadas (promedio, mediana, desviación estándar) y genera cinco reportes analíticos en formato HTML.
El sistema simula las fases iniciales de un compilador: lectura estructurada de archivos, tokenización de datos, validación de campos y generación de salidas formateadas. Incluye un menú interactivo en consola con manejo de excepciones para entradas inválidas.
Funcionalidades principales:

Carga de archivos estudiantes.lfp, cursos.lfp y notas.lfp
Cálculo de promedio, mediana, desviación estándar por curso
Ranking de los 10 mejores estudiantes
Detección de cursos con mayor índice de reprobación
Análisis estadístico por carrera y distribución por semestre
Generación de 5 reportes HTML con formato profesional
